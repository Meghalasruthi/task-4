
# Task 4: Firewall Setup and Configuration (Windows)

## Objective
To configure and test basic firewall rules to allow and block network traffic using Windows Defender Firewall.

## Tools Used
- Windows Defender Firewall
- Command Prompt

## Steps Performed
1. Opened Windows Defender Firewall.
2. Viewed existing inbound firewall rules.
3. Created a new inbound rule to block Telnet traffic on port 23.
4. Tested the blocked rule using command prompt.
5. Created a new inbound rule to allow SSH traffic on port 22.
6. Verified the firewall rules.
7. Removed the test block rule to restore original firewall state.

## Outcome
Gained basic understanding of firewall configuration, inbound rules, port-based traffic filtering, and network security concepts.
